# Static Images Catalog

Non-device images for buttons, tiles, arrows, and media controls. Located in `Web Assets/images/controls/static/`.

These images do NOT have state variants — they are static decorative or layout elements.

## Tiles

Background tiles for grouping elements into visual sections.

Each tile has Header (top), Body (middle), and Footer (bottom) pieces that stack vertically.

### Sizes

| Size | Approximate Width |
|------|-------------------|
| Small | ~150px |
| Medium | ~250px |
| Large | ~350px |
| X-Large | ~450px |

### Tile Parts

| ImageFileName Pattern | Description |
|-----------------------|-------------|
| `Light Tile Header {Size} 2x` | Top rounded corners |
| `Light Tile Body {Size} 2x` | Middle (repeatable) |
| `Light Tile Footer {Size} 2x` | Bottom rounded corners |

Example set for a medium tile:
- `Light Tile Header Medium 2x`
- `Light Tile Body Medium 2x`
- `Light Tile Footer Medium 2x`

1x variants also available.
